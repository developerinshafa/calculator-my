Calculator App

A simple and responsive Calculator Application built using HTML, CSS, and JavaScript.
This calculator supports basic arithmetic operations with keyboard support and a clean modern UI.

 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🔢 Decimal Support
⌨️ Keyboard Support
🧹 Clear All (C)
🗑️ Clear Entry (CE)
⌫ Backspace Function

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
